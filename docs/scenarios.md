# QA シナリオ

## シナリオ: ZIPファイルアップロード（正常系）
- 前提: ブラウザでアプリを開いている
- 操作: Garmin Connect からエクスポートした ZIP ファイルをアップロードボタンまたはドロップゾーンに渡す
- 期待結果: ファイルが受け付けられ、変換処理が開始される（進捗表示 or スピナー）

## シナリオ: 変換・ダウンロード（正常系）
- 前提: 有効な Garmin ZIP がアップロードされた状態
- 操作: 変換が完了するのを待つ（自動）
- 期待結果: 変換済み ZIP のダウンロードが自動的に開始される、またはダウンロードボタンが現れる

## シナリオ: 変換後ZIPの中身確認
- 前提: ダウンロード済みの変換 ZIP
- 操作: ZIP を展開して内容を確認する
- 期待結果:
  - `activities.csv` が存在する（ヘッダー付き、クリーンなCSV）
  - `sleep.csv` が存在する
  - `daily_health.csv` が存在する
  - `laps.csv` が存在する
  - `prompt_template.txt`（AIへの使い方ガイド）が同梱されている

## シナリオ: 非ZIPファイルのアップロード（異常系）
- 前提: ブラウザでアプリを開いている
- 操作: ZIP 以外のファイル（例: `.jpg`, `.pdf`）をアップロードしようとする
- 期待結果: エラーメッセージが表示される（例:「ZIPファイルを選択してください」）

## シナリオ: Garmin ZIP 以外のZIPアップロード（異常系）
- 前提: ブラウザでアプリを開いている
- 操作: Garmin 形式でない ZIP（空の ZIP や無関係ファイル）をアップロードする
- 期待結果: 対象 CSV が見つからない旨のエラーまたは警告が表示される

## シナリオ: スマートフォンでの操作性
- 前提: iPhone / Android のモバイルブラウザでアプリを開く
- 操作: ファイルピッカーでZIPを選択しアップロードする
- 期待結果:
  - ボタン・テキストが読みやすいサイズで表示される
  - タップ操作でファイル選択・ダウンロードが完結する（ドラッグ不要）
  - UI 表記が英語で統一されている

## シナリオ: プライバシー・通信確認
- 前提: ブラウザのネットワークタブを開いた状態でアプリを使用
- 操作: ZIPをアップロードして変換を実行する
- 期待結果: ファイルデータを外部サーバーに送信するリクエストが発生しない（ブラウザ完結）

## シナリオ: 大容量ZIPの処理
- 前提: 数百MB〜1GB 以上の Garmin ZIP
- 操作: アップロード・変換を実行する
- 期待結果: クラッシュせずに処理完了する（進捗表示があると尚良い）
